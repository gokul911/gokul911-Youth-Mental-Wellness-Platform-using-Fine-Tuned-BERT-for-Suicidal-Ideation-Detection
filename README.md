## 🧠 Project Description
This project focuses on detecting suicidal ideation in textual data using a fine-tuned BERT model. It aims to provide a privacy-first mental wellness platform with AI-driven insights and early intervention mechanisms.

## ⚙️ Tech Stack
- BERT (NLP Model)
- Scikit-learn

## 📊 Features
- Suicidal ideation detection
- Model comparison (BERT vs ML models)
- Confusion matrix & performance analysis
- Privacy-first architecture
